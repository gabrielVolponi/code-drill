About the Project

This website was created to help people who are learning JavaScript practice in a personalized and dynamic way. The idea is simple: you choose the difficulty level (easy, medium, or hard), select the topic you want to practice (such as functions, arrays, objects, loops, etc.), and the system automatically generates a tailored exercise for you.

To achieve this, we use the OpenAI API, which allows us to generate smart and varied prompts based on the selected parameters. The website's interface is built with JavaScript, Tailwind CSS, and JSON for communication with the API, ensuring a modern, fast, and adaptable experience.
